What is "Git"?
---------------
Git is a protocol (rules) to control the source code. The Git client tools allow us to save, update, undo/redo, distribute, merge our source code (projects).  "Git" protocol and tool is introduced by the creator of "Linux Operation System"; Linus Torvald.
